## 常見錯誤及解法
| 錯誤 | 解法 |
|------|------|
| `X is not defined` | 檢查 tab 函數定義有沒有該 prop |
| `export named 'default'` | 加 `export default` |
| `useState is not defined` | 頂部加 `import { useState, useEffect } from 'react'` |
| 400 Bad Request | `useEffect` 加 `if(companyId)` 保護 |
| 406 Not Acceptable | `.single()` 改 `.maybeSingle()` |
| companyId is not defined | tab 函數定義加 `companyId` prop |
| employee_visa 400 | 用 `employees!inner` join 而非直接 eq company_id |
