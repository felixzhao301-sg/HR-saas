# HR SaaS 項目進度記錄

## 2026-04-20（今天）

### ✅ 完成功能

#### 1. 員工模塊補充
- `join_date`（入職日期）字段加入 employees 表
- `work_email`（公司電郵）字段加入 employees 表
- EmployeesTab UI 更新：表單、詳情頁、批量上傳均支持新字段

#### 2. 薪資模塊（全新）
**數據庫（5張新表）**
- `company_payroll_settings` — 公司薪資設置
- `payroll_working_days` — 每月工作天數參考
- `payroll_runs` — 薪資批次（每月一次）
- `payroll_records` — 每員工薪資記錄
- `commissions` — 佣金模塊

**計算引擎 `src/utils/payroll.js`**
- CPF 自動計算（2026年最新比率）
- 按員工年齡、國籍、PR年份自動選比率
- OW 上限 $8,000 / AW 年度上限 $102,000
- SDL、FWL、CDAC/MBMF/SINDA 自動計算
- 無薪假 MOM 公式自動扣款
- Director Fee 獨立處理（計Gross不交CPF）
- Other Adjustments 多行可正可負

**PayrollTab（薪資管理主模塊）**
- 建立每月薪資批次，自動為全體全職員工生成記錄
- 實時計算預覽（展開員工行即顯示）
- Draft → Confirmed → Locked 流程
- Lock 時計算 YTD 快照
- 只顯示 full_time 員工（part_time 接口預留）

**CommissionTab（佣金管理）**
- 按月輸入員工佣金
- draft → confirmed 流程
- 確認後自動帶入薪資計算

**三個報告（打印/存PDF）**
- 📄 工資單（PayslipModal）— 對應現有紙質工資單格式，文件名：`NAME_Payslip_Apr2026.pdf`
- 🏦 CPF 匯總（CPFReportModal）— 對應 CPF EZPay 官方格式
- 📋 工資匯總（PayrollSummaryModal）— 全公司當月，含銀行轉帳明細

#### 3. SettingsTab 薪資設置
- 每週工作天數選擇（5/5.5/6天）
- Logo 上傳（Supabase Storage）
- CPF 提交號碼設置
- 工資單頁腳文字設置

#### 4. 權限新增

```sql
payroll.view / edit / lock / report
commission.edit
```
已為 YLL 公司的 hr_admin / hr_staff / super_admin 插入

#### 5. UserManagementTab 改進
- 創建帳號成功後顯示郵件發送狀態 toast
- 重設密碼郵件發送確認

#### 6. i18n 語言文字拆分
