郵件架構（重要）
所有郵件統一走 Edge Function：
App → supabase.functions.invoke('send-email') → Resend API → 收件人
Edge Function 支持模板：

leave_submitted — 年假申請通知批准人
leave_approved — 年假批准通知員工
leave_rejected — 年假拒絕通知員工
password_reset — 忘記密碼重設連結

⚠️ 重要設定：

Resend API Key 存在 Supabase Edge Function Secrets（RESEND_API_KEY）
Edge Function JWT 驗證已在 Dashboard 關閉（Verify JWT = OFF）
Supabase SMTP 設定了 Resend SMTP（備用，但主要走 Edge Function）
發件人：HR SaaS <onboarding@resend.dev>（免費版）
